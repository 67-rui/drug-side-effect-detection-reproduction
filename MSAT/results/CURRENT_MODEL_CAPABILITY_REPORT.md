## 2. 当前模型的主要任务

当前 MSAT 模型面向中药材不良反应预测任务，核心目标是判断：

```text
某味中药材 CMM 是否可能与某种不良反应 ADR 存在关联
```

模型输入可以理解为一个中药材与一个不良反应的组合，例如：

```text
艾叶 -> 药物性肝损伤
虎杖 -> 呕吐
刺五加 -> 耳鸣
```

模型输出为 0 到 1 之间的风险关联分数。该分数表示该组合在当前模型看来是否值得作为药物警戒信号进一步关注。

需要注意的是，该分数不是临床发生率，也不是因果证明。它更适合被理解为风险信号的排序分数。

## 3. 当前模型能完成的工作

### 3.1 中药材-不良反应关联预测

模型可以对任意候选的 CMM-ADR 组合进行打分，用于判断该组合是否可能存在潜在安全风险。

例如：

```text
输入：某味中药材 + 某个 ADR
输出：该组合的风险关联分数
```

这一能力适合用于药物警戒场景中的初筛工作。它可以帮助研究者从大量候选组合中筛出更值得查阅文献、数据库或进行人工复核的对象。

### 3.2 某一味中药材的 ADR 风险排序

模型可以固定一味中药材，对全部 ADR 逐一打分，并生成候选不良反应列表。

例如：

```text
输入：艾叶
输出：艾叶相关 ADR 的排序列表
```

这种结果可以用于构建某味中药材的潜在风险画像。实际使用时，应将高分结果作为进一步核验的候选，而不是直接作为安全结论。

### 3.3 某一类 ADR 的相关中药材筛查

模型也可以反向使用：固定某个 ADR，查看哪些中药材与该 ADR 的关联分数较高。

例如：

```text
输入：药物性肝损伤
输出：与该 ADR 关联分数较高的中药材列表
```

该能力适合用于围绕某类不良反应进行风险来源筛查，例如肝损伤、胃肠道反应、心血管反应、皮肤反应等。

### 3.4 基于异构图的机制辅助推断

当前模型不是只依赖中药材与 ADR 的报告共现关系。它还利用了异构图中的多类关系：

```text
中药材 -> 化学成分
化学成分 -> 蛋白靶点
蛋白靶点 -> ADR
中药材 -> ADR
```

因此，模型在预测某个 CMM-ADR 关联时，可以间接利用成分、靶点和不良反应之间的结构信息。

这使得模型不完全依赖已有报告记录。对于历史报告较少的中药材，模型仍可能通过成分和靶点层面的信息给出风险推断。

### 3.5 冷启动场景下的预测能力

冷启动指训练阶段几乎没有见过某些中药材的 CMM-ADR 关系，但测试阶段仍需要对这些中药材进行预测。

当前复现结果中，FAERS-only 训练、文献集测试的冷启动评估显示模型具有一定泛化能力：

```text
AUC 约为 0.933
```

并且在该设置下，MSAT 优于 GAT、HGT、Simple-HGN 等图神经网络基线。

这说明当前模型不是简单记忆训练集中出现过的中药材-ADR 组合，而是能够利用图结构和节点特征进行一定程度的外推。

## 5. 不平衡场景结果

在 1:10 正负样本不平衡实验中，当前 MSAT 结果为：

| 指标 | MSAT |
|---|---:|
| AUC | 0.8710 |
| AUPRC | 0.5917 |
| F1 | 0.5604 |
| MCC | 0.5180 |

该结果接近论文中的 1:10 不平衡实验结果，说明模型在更接近真实药物警戒稀疏场景时仍有一定稳定性。

不过在当前复现中，XGB 的 AUC 和 AUPRC 略高于 MSAT。因此，这部分不能简单表述为“MSAT 在所有指标上均第一”。更稳妥的说法是：MSAT 在不平衡场景下保持较强竞争力，但全模型排序与论文存在小幅差异。

## 6. 消融实验结果

当前已完成 MSAT 主要模块的消融实验，包括：

- 去除 ESA-Gate
- 去除 HSP-Layer
- 去除 HCI-Module

整体结果说明，完整模型和各变体之间的差异较小，但完整模型仍处于较高水平。

需要注意的是，当前消融结果的趋势与论文不完全一致。例如，`wo_esa` 的 AUC 略高于 Full，差异很小，但说明不同模块的贡献排序在当前复现实验中没有完全复刻论文结果。

因此，消融部分可以作为已有实验结果保留，但在报告或论文中需要如实说明趋势差异。

- 文献支持是否需要人工逐条核验；
- MedDRA 术语与文献描述之间是否存在同义词或层级映射差异。

## 8. 当前模型的适用范围

当前模型适合用于以下工作：

- 中药材潜在 ADR 风险信号发现；
- 某味中药材的 ADR 候选列表生成；
- 某类 ADR 相关中药材筛查；
- FAERS 信号与机制知识图谱联合分析；
- 后续文献验证、数据库验证和机制路径分析的前置筛选。

当前模型更适合作为药物警戒研究中的风险排序工具，而不是临床结论工具。

## 9. 当前模型的限制

当前模型仍有以下限制。

第一，模型不能证明因果关系。高分结果只表示该组合值得关注，不代表某味中药材一定导致某种不良反应。

第二，模型不能估计真实发生率。FAERS 缺少可靠的暴露人群基数，因此无法据此计算真实风险概率。

第三，模型目前主要面向单味中药材，不足以处理复方配伍、剂量、炮制方式、患者基础疾病、合并用药等复杂因素。

第四，外部证据验证仍不充分。尤其是 Table 5 部分，目前尚未达到论文报告的外部支持率。

当前 MSAT 模型已经基本达到论文主实验中的预测效果，可以作为后续科研工作的基础模型。

## 11. 后续可拓展方向

### 方向一：面向不完整标签的 PU-MSAT

当前实验中，未观察到的 CMM-ADR 组合通常被当作负样本。但在药物警戒数据中，未观察到不等于真实不存在。很多潜在风险可能只是没有被报告、没有被记录或没有被文献验证。

因此，可以将当前二分类任务扩展为 Positive-Unlabeled Learning：

```text
已知 CMM-ADR 关联 = 正样本
未观察到的 CMM-ADR 组合 = 未标注样本
```

这样可以减少把潜在风险误当作负样本的问题。该方向与 FAERS 和中药 ADR 数据的标签不完整特征高度相关，适合在 1:10 不平衡场景、冷启动场景和 Top-K 外部验证中进行比较。

该方向可形成的研究主题为：

```text
PU-MSAT: 面向不完整药物警戒标签的正样本-未标注异构图学习
```

### 方向二：机制路径增强的可解释风险重排序

当前模型可以给出 CMM-ADR 风险分数，但对具体机制路径的解释仍不够充分。尤其是 Table 5 外部资料辅助验证尚未完成，说明仅依赖模型分数进行 Top-K 排序还不够稳妥。

可以在当前 MSAT 分数基础上加入机制路径证据，对候选结果进行二次排序：

```text
最终排序分数 =
MSAT 预测分数
+ FAERS 证据强度
+ CMM -> Compound -> Target -> ADR 机制路径支持
+ 外部数据库或文献证据
```

该方向的目标不是只提高 AUC，而是提高高置信预测结果的可解释性和外部证据支持率。

该方向可形成的研究主题为：

```text
机制路径增强的中药材不良反应可解释预测与重排序
```

这两个方向都可以在当前模型基础上展开。前者主要解决负样本不可靠的问题，偏建模方法创新；后者主要解决结果解释和外部验证不足的问题，偏药物警戒应用创新。

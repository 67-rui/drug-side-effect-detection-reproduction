# PU-XMSAT 口头汇报稿

**日期：** 2026-06-27
**用途：** 3-5 分钟导师沟通或组会口头汇报。该稿基于当前已验证结果，不新增实验结论。

## 1. 开场概括

老师，我目前的工作分成两部分：第一部分是复现 MSAT 论文的主实验，第二部分是在复现基础上针对负样本噪声问题做了一个 PU Learning 扩展，也就是 PU-XMSAT。

现在主实验复现已经基本完成，当前审计结果是 `issues: []`。我采用的主实验协议是 `2026-06-23-val-test-edge-hidden`，也就是在训练图里隐藏验证集和测试集的正样本边，避免信息泄漏。复现得到的 MSAT 主基线是 AUC 0.979271、AUPRC 0.977095、F1 0.931451、MCC 0.862520。

## 2. 需要特别说明的复现边界

这里有一个边界需要先说明：Table 5 和 Table 6 我现在不把它们作为主实验复现结论。它们更像是外部验证和解释性案例。

Table 5 依赖作者未完全公开的候选生成流程、预测 checkpoint 和证据筛选规则；Table 6 又依赖 Table 5 的候选。因此我现在的表述是：主实验与多数核心性能结果已经复现，但 Table 5/6 在公开材料下无法论文等价复现。这个边界我会在论文或汇报中明确写出来。

## 3. 为什么提出 PU-XMSAT

在复现过程中，我发现一个比较关键的问题：原始二分类训练会把很多未观测到的中药-不良反应配对当成负样本。但在药物警戒场景里，“未观测”并不一定代表“无关联”，它可能只是没有被报告、没有被充分暴露，或者受到适应症偏倚、合并用药、报告偏倚等因素影响。

所以我做了 PU-XMSAT。它保留 MSAT 的图结构、10 折划分和训练后端，只改变监督样本构造方式：把配对拆成观测正样本、可靠负样本和未标注样本。未标注样本仍然参与训练，但降低权重，从而减少“未观测直接当负样本”带来的标签噪声。

## 4. 当前 PU-XMSAT 的主要结果

我完成了 corrected random-cache 和 full-positive pair budget 下的主要实验。当前最强设置是 `hybrid` reliable-negative sampling，使用 `66,015` pairs per fold，默认权重是 `unlabeled_weight=0.2`、`reliable_negative_weight=0.8`。

random PU-XMSAT 可以达到与 MSAT 接近的水平，但统计上没有稳定显著优势。因此我把 random 作为可行性和消融对照。

hybrid PU-XMSAT 是当前最好的结果。seed=2026 时，AUC 0.980420、AUPRC 0.977929、F1 0.935064、MCC 0.868409；相对 MSAT 的 paired test 四个指标都过了 0.05。seed=1337 复跑时，AUC 0.980392、AUPRC 0.977983、F1 0.934767、MCC 0.868331；其中 AUC、F1、MCC 仍然显著优于 MSAT，AUPRC 是正向但边界显著，p=0.056348。

两个 hybrid seed 之间非常接近，AUC range 只有 0.000028，AUPRC range 只有 0.000054，F1 range 0.000297，MCC range 0.000078。这说明结果不是单一随机种子的偶然现象。

## 5. 权重敏感性和当前结论

我还做了 PU 权重敏感性。固定 reliable-negative weight 为 0.8，比较 unlabeled weight 0.1、0.2、0.4。结果显示 u0.1 整体变弱，u0.4 的 AUC 接近但 F1 和 MCC 略弱，u0.2/rn0.8 是目前 ranking 指标和阈值指标之间比较平衡的设置。

所以我现在比较稳妥的结论是：PU-XMSAT 不是全面宣称优于 MSAT，而是说 full-positive hybrid PU-XMSAT 在两个 seed 下对 AUC、F1、MCC 有稳定提升，AUPRC 有一致正向趋势但提升幅度较小。这个结论边界我会保留。

## 6. 下一步计划

下一步我不打算继续重复长时间训练，除非发现新的代码或数据问题。更有价值的是把这部分整理成论文内容：

1. 方法部分重点写“未观测不等于负样本”、可靠负样本选择和未标注样本降权训练；
2. 结果部分整理 MSAT baseline、random PU-XMSAT、hybrid PU-XMSAT 两个 seed、paired test 和权重敏感性；
3. 讨论部分保留 Table 5/6 的复现边界，并把它们转为外部证据和解释性分析的后续方向；
4. 后续如果继续扩展模型，可以接导师建议，往因果混杂控制、关键子图解释、SHAP/注意力解释和更严格负样本策略方向推进。

## 7. 一句话总结

一句话总结就是：我已经完成 MSAT 主实验复现，并在复现基础上把“未观测样本直接当负样本”的问题转化为 PU Learning 改进；当前 full-positive hybrid PU-XMSAT 在两个 seed 下表现稳定，AUC、F1、MCC 相对复现 MSAT 基线有稳定提升，下一步重点从继续训练转向论文写作、外部证据和解释性分析。

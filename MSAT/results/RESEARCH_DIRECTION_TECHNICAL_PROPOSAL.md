老师您好：

根据您对当前模型后续改进方向提出的建议，我对 MSAT 论文复现进展、复现过程中暴露出的问题，以及下一步可继续深入的研究方向进行了整理。整体判断是：当前项目已经具备从“论文复现”进入“基于复现的模型改进”的基础。后续不宜只停留在继续追求论文所有辅助案例表完全一致，而应围绕复现过程中发现的核心问题，形成一个有明确科学问题、有技术增量、也能够通过实验验证的研究方案。

2. 复现过程中暴露出的关键问题；
3. 拟定的后续研究方向与技术路线；
4. 后续实验设计、预期成果与需要老师进一步指导的问题。

## 一、当前复现工作的基础

目前项目已经完成了 MSAT 模型主实验流程的复现与核查，主要包括数据读取、特征构建、模型训练、交叉验证、评价指标汇总以及论文协议对齐检查。从现有实验结果看，模型核心指标已经接近论文主实验结果，说明当前复现并不是停留在“代码可以运行”的层面，而是已经基本重建了原论文的主要实验效果。

当前主实验复现指标如下：

| 指标 | 当前复现结果 | 说明 |
| --- | ---: | --- |
| AUC | 0.9793 | 与论文主实验表现接近 |
| AUPRC | 0.9771 | 说明正负样本区分能力较强 |
| F1 | 0.9315 | 综合精确率与召回率表现较好 |
| MCC | 0.8625 | 分类结果整体较稳定 |

基于上述结果，我认为当前项目已经具备继续开展模型改进的基础。主实验层面可以作为后续工作的稳定基线，后续新增方法也可以在相同协议下与原始 MSAT 进行对比。

同时，在复现过程中也发现了一个需要明确区分的问题：论文中的主实验指标与 Table 5、Table 6 等案例或辅助分析表格并不完全等价。主实验主要用于证明模型整体预测性能，而 Table 5、Table 6 更多用于案例展示、机制解释或补充佐证。当前公开材料能够较好支持主实验复现，但对部分辅助案例表的完整生成流程、候选筛选规则和证据筛选标准公开不足，因此不适合将这些辅助表格作为当前复现是否成功的唯一判断标准。

## 二、复现过程中暴露出的关键问题

### 1. “未观测”不等于“无关联”

在药物不良反应预测任务中，正样本通常来自已经被数据库、文献或报告系统记录的药物-副作用关联；负样本则往往由未记录的药物-副作用组合采样得到。然而，在真实医学数据中，未记录并不一定代表真实无关联，也可能只是尚未被发现、尚未被报告、数据库尚未收录，或因为暴露人群不足而暂时没有观察到。

如果直接将所有未观测组合作为负样本，模型训练可能面临以下问题：

1. 一部分潜在正样本会被错误标注为负样本；
2. 模型可能学习到数据库收录偏差，而不是真实的药物-副作用机制；
3. 对证据稀疏但实际可能存在风险的组合，模型可能被训练成低分；
4. 后续案例解释中可能混入由标签噪声导致的伪相关。

因此，我认为负样本策略是当前任务中最直接、最值得优先推进的改进方向。它既来自复现过程中发现的真实问题，也与药物副作用预测任务本身的数据特点高度相关。

### 2. 模型预测能力较强，但解释性仍然不足

MSAT 通过多源特征和注意力机制建模药物、成分、靶点、疾病、副作用之间的关系，能够取得较高预测性能。但从后续研究角度看，仅有预测分数仍然不足，尤其是在中药不良反应预测场景中，还需要回答以下问题：

1. 某个药物-副作用预测主要由哪些成分、靶点或路径支持？
2. 注意力权重能否稳定反映模型判断依据？
3. 单个成分、靶点或机制路径对预测分数的贡献能否被量化？
4. 模型输出能否转化为便于医学讨论和人工复核的解释结果？

因此，后续需要在原始 MSAT 基础上增加机制子图提取和贡献度量化模块，使模型从“能预测”进一步走向“能解释”。

### 3. 混杂因素和报告偏倚需要纳入研究视野

您提到的因果图和因果论方向非常重要。药物副作用数据天然受到合并用药、适应症偏倚、报告偏倚、暴露人群差异等因素影响。例如，某个副作用可能并不是由单一药物直接导致，而是与适应症、联合用药、人群特征或报告概率有关。

不过从当前数据条件看，本项目主要基于知识图谱和药物-副作用关联数据，暂时还缺少个体级用药记录、时间顺序、合并用药、适应症、人群统计变量和暴露基数。因此，如果直接把严格因果推断作为第一阶段核心方法，可能会受到数据支撑不足的限制。

我的建议是：当前阶段先将因果图作为问题建模和偏倚分析框架，用于明确哪些混杂因素可能影响模型；等后续能够引入 FAERS、电子病历或真实世界用药数据后，再进一步开展因果校正或因果表示学习。

### 4. 外部证据可以辅助验证，但不应直接替代标注

关于使用 DeepSeek 等大模型辅助副作用评估，我认为这一方向适合作为“证据辅助层”，而不是直接作为训练标签来源。大模型可以帮助完成文献摘要、证据筛选、候选关系解释和人工复核辅助，但如果直接把大模型判断作为标签，可能引入幻觉、知识污染和循环验证问题。

因此，更稳妥的做法是采用“检索-摘要-分级-人工复核”的流程：先从 PubMed、SIDER、DrugBank、FAERS 或说明书中检索证据，再利用大模型辅助归纳，最后保留来源和人工审核结论。

## 三、拟定研究方向

结合当前复现基础和您提出的建议，我拟将后续研究方向确定为：

**基于不完整标签学习与机制解释的中药不良反应预测模型改进研究。**

该方向的核心思路是：在已经复现的 MSAT 模型基础上，针对药物副作用数据中“未观测关系不等于真实负样本”的问题，引入可靠负样本选择和 PU Learning 方法，降低标签噪声；同时结合注意力权重、子图提取和扰动贡献分析，提高模型对关键成分、靶点和机制路径的解释能力；最后通过外部文献和数据库证据对高置信预测结果进行辅助验证。

整体上，该方向可以概括为三个层次：

1. **训练层面：** 改进负样本构建方式，降低未标注样本误作负样本带来的噪声；
2. **解释层面：** 提取关键机制子图，量化成分、靶点和路径贡献；
3. **验证层面：** 引入外部证据分级，提高候选预测结果的可信度。

## 四、技术路线设计

### 1. 可靠负样本选择与 PU Learning

原始 MSAT 任务中，负样本通常从未观测药物-副作用组合中采样。但在真实医学场景中，未观测组合应更合理地看作“未标注样本”，而不是确定的负样本。因此，后续可以将训练数据重新划分为：

1. 已观测正样本；
2. 高可信负样本；
3. 未标注样本。

其中，高可信负样本可以从以下角度筛选：

1. 与目标副作用缺少结构关联的药物组合；
2. 与已知相关药物相似度较低的组合；
3. 基线模型预测分数稳定较低的组合；
4. 缺少共同靶点、共同疾病或机制路径支持的组合。

在此基础上，可以尝试两类训练方法：

1. **可靠负样本 MSAT：** 用筛选出的高可信负样本替代随机负样本；
2. **PU-MSAT：** 将未观测组合视为未标注样本，通过加权损失或 PU Learning 风险估计进行训练。

第一阶段建议优先实现加权 PU Loss，因为它与现有 MSAT 训练流程兼容性较高，工程风险较低，也便于做消融实验。

### 2. 关键机制子图提取与贡献度量化

在解释性方面，建议围绕单个高置信药物-副作用预测结果，提取其相关机制子图。子图可以重点关注以下路径：

1. 药物-成分-靶点-副作用；
2. 药物-成分-靶点-疾病-副作用；
3. 药物-靶点-副作用；
4. 药物-疾病-副作用。

贡献度可以从两个角度量化：

1. **注意力权重：** 利用模型内部注意力分数识别关键节点和边；
2. **扰动贡献：** 分别屏蔽某个成分、靶点或路径，观察预测分数下降幅度。

如果后续条件允许，还可以进一步加入 SHAP-style 的局部贡献分析，用于更系统地量化单个成分或靶点对预测结果的影响。

### 3. 外部证据辅助验证

对于模型预测出的高置信候选药物-副作用关系，可以建立外部证据分级体系。建议将证据分为四类：

| 等级 | 含义 | 用途 |
| --- | --- | --- |
| A | 数据库、说明书或权威资料直接记录该副作用 | 强证据 |
| B | 文献直接报告该药物与该副作用相关 | 强证据 |
| C | 文献支持相关成分、靶点或机制路径 | 间接机制证据 |
| D | 暂无明确证据或证据不一致 | 保留候选 |

该流程可以用于升级原论文 Table 5 类型的案例分析。相比只展示预测分数和文献支持，后续可以形成“预测分数 + 机制子图 + 贡献度 + 外部证据等级”的案例表，使结果更适合展示和讨论。

### 4. 因果图方向的阶段性安排

因果方向建议分阶段推进：

1. **当前阶段：** 建立因果图框架，明确合并用药、适应症偏倚、报告偏倚和暴露人群差异可能如何影响模型预测；
2. **后续阶段：** 如果能够获得更细粒度真实世界数据，再考虑倾向评分、逆概率加权、因果图约束表示学习等方法。

这样既能回应因果建模的重要性，也能避免当前数据不足时过早承诺严格因果结论。

## 五、实验设计

后续实验建议保留原始 MSAT 作为基线，并在相同评价协议下逐步加入改进模块。建议设置以下模型：

| 模型 | 目的 |
| --- | --- |
| 原始 MSAT | 已复现主实验基线 |
| MSAT + 可靠负样本 | 验证负样本筛选是否有效 |
| MSAT + PU Loss | 验证 PU Learning 是否降低标签噪声 |
| MSAT + 可靠负样本 + PU Loss | 验证两类方法组合效果 |
| 上述模型 + 机制子图解释 | 验证解释结果是否稳定且具有医学意义 |
| 上述模型 + 外部证据分级 | 验证高置信候选关系的外部支持情况 |

主实验指标继续沿用论文指标，包括 AUC、AUPRC、F1、MCC、Precision 和 Recall，以保证与原始 MSAT 可比。

除常规预测指标外，建议增加以下分析：

1. 不同负样本比例下的性能变化；
2. 高频和低频副作用上的分组表现；
3. 高相似药物群体中的误判情况；
4. 关键子图移除后预测分数下降情况；
5. 高置信案例的外部证据支持比例。

## 六、预期创新点

本方案预期可以形成以下几个创新点：

1. **任务设定层面：** 将药物副作用预测中的未观测关系重新建模为未标注样本，而不是简单负样本；
2. **训练策略层面：** 提出可靠负样本选择与 PU Learning 融合策略，降低不完整标签带来的噪声；
3. **解释性层面：** 基于注意力权重和扰动贡献提取关键机制子图，量化成分、靶点和路径贡献；
4. **证据验证层面：** 建立外部证据分级流程，使模型预测结果更便于人工复核和医学讨论；
5. **复现驱动层面：** 从论文复现过程中发现问题，并将复现缺口转化为可继续研究的模型改进方向。

## 七、近期一周工作安排

如果老师认可上述方向，我计划先在一周内完成一个最小可验证闭环，不立即铺开所有扩展工作。近期重点是先验证“可靠负样本与 PU Learning 是否值得作为主线继续推进”，并同步准备少量解释性案例。

本周内拟完成以下工作：

1. 固化原始 MSAT 主实验结果，作为后续所有改进实验的对照基线；
2. 设计并实现一版可靠负样本筛选策略，与原始随机负采样进行初步对比；
3. 实现一版加权 PU Loss，在小规模实验中验证训练流程是否稳定；
4. 选择少量高置信预测案例，初步提取相关成分、靶点和机制路径；
5. 汇总一份一周内的初步实验结果，重点说明该方向是否具备继续深入的价值。

这一周的目标不是一次性完成全部模型改进，而是先判断主线是否成立：如果可靠负样本或 PU Learning 能够带来性能、鲁棒性或案例可信度上的改善，则继续扩展完整实验；如果提升不明显，则及时调整为以解释性分析或外部证据验证为主要突破口。

## 八、风险与应对

| 风险 | 可能影响 | 应对策略 |
| --- | --- | --- |
| PU Learning 对整体指标提升不明显 | 难以形成单纯性能提升 | 增加标签噪声鲁棒性、低频副作用和案例质量分析 |
| 可靠负样本筛选过严 | 可用训练样本不足 | 设置多档阈值，做敏感性分析 |
| 注意力解释不稳定 | 解释结果可信度不足 | 结合扰动贡献和多随机种子稳定性评价 |
| 外部证据不完整 | 部分候选关系难以验证 | 采用证据分级，不将证据缺失等同于关系不存在 |
| 因果方向数据不足 | 难以开展严格因果推断 | 当前阶段先做因果图框架，后续引入真实世界数据后再扩展 |

## 九、希望老师进一步指导的问题

基于以上方案，我希望后续请老师重点帮我把关以下几个问题：

1. 是否认可将“未观测不等于负样本”作为后续研究的主问题；
2. 负样本策略与 PU Learning 是否适合作为当前阶段的主要创新点；
3. 解释性部分更应强调成分贡献、靶点贡献，还是机制路径贡献；
4. 因果图方向目前作为理论框架是否合适，是否需要提前引入更严格的因果方法；
5. 外部证据辅助验证是否需要优先使用 PubMed、FAERS、说明书，还是结合其他中医药相关数据库。

## 十、结论

总体来看，当前 MSAT 主实验复现已经为后续研究提供了比较可靠的基线。下一阶段更适合围绕复现中暴露出的关键问题推进模型改进，而不是单纯继续追求所有论文辅助表格的逐项复刻。

我目前判断，最适合继续推进的路线是：以可靠负样本选择和 PU Learning 解决不完整标签问题，以机制子图和贡献度分析提升模型解释性，以外部证据分级增强案例可信度，并将因果图作为混杂因素分析和后续扩展方向。

该路线能够在现有复现基础上逐步实现、逐步验证，也便于根据初步实验结果及时调整研究重点。如果这个方向整体可行，我下一步将优先实现可靠负样本策略与 PU Loss，并以原始 MSAT 复现结果作为基线开展对比实验。
